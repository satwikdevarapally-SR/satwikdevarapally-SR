## 📈 My Projects

### 🏏 IPL Match Analysis
Analyzed **756 IPL matches** to find patterns in team wins, venues, and toss impact.
- **Tools:** Python, Pandas, Matplotlib  
- **Key Finding:** Toss impact is ~52% (mostly luck!)
- **Repo:** [IPL-Match-Analysis](https://github.com/satwikdevarapally-SR/IPL-Match-Analysis)

### ⚽ World Cup 2026 Predictor

- **Accuracy:** 67.65%  
- **Model:** Random Forest Classifier
- **Tools:** Scikit-learn, Pandas
- **Repo:** [WC-2026-Predictor](https://github.com/satwikdevarapally-SR/WC-2026-Predictor)
